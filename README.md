# Telco Customer Churn Prediction - Milestone 2

## Project Objective
The objective of this project is to prepare the Telco Customer Churn dataset for machine learning by performing data preprocessing and exploratory data analysis (EDA).

## Folder Structure

data/
- Telco-Customer-Churn.csv (Original Dataset)

notebook/
- Milestone2_Data_Preprocessing.ipynb

results/
- EDA_graphs/
- final_data/

## Tasks Performed
- Imported the dataset
- Performed basic data exploration
- Checked missing values and duplicates
- Cleaned the dataset
- Created EDA graphs
- Split the data into training and testing sets
- Saved X_train, X_test, y_train and y_test

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Output
The final processed datasets are available in the `results/final_data` folder.
